# Socialmedia-database

### Project Overview
The Social Media Database Management System is a DBMS project developed using SQL. The project simulates the basic functionality of a social media platform by managing user profiles, posts, comments, likes, and user connections.

### Database Tables
The project consists of the following tables:
- Users
- Posts
- Comments
- Post_Likes
- Connections

### SQL Concepts Implemented
- Create database
- Create table
- Primary key
- Foreign key
- Unique Constraints
- Not null Constraints
- Insert
- Select
- Where
- Like
- Order by
- Group by
- Count
- Inner join
- Left join
- Right join

### Project Outcome
The database successfully manages users, posts, comments, likes, and connections while maintaining data integrity through relational database concepts.
